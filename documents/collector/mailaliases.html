Работа со списками рассылки


Технически, список рассылки это почтовый адрес без почтового ящика. Вместо почтового ящика с ним связан список других почтовых адресов (которые могут быть расположены как локально, так и в Интернет). При поступлении письма на адрес списка рассылки, почтовый сервер не сохраняет его, а пересылает по всем адресам этого списка.

Создание списка рассылки


1. Найдите в дереве домен, внутри которого хотите создать список рассылки.

2. Нажмите на нем правой кнопкой мыши и выполните команду "Новый список рассылки" из контекстного меню.



3. Заполните форму параметров списка рассылки



В поле "Имя" указывается первая часть адреса списка рассылки (вторая часть это имя домена, в котором он создается).

В поле "Яшик для спама" указывается почтовый ящик, в который будут пересылаться письма, которые почтовый сервер классифицирует как спам. Это обычный почтовый ящик на сервере, который предварительно должен быть создан.

4. Нажмите кнопку "ОК"

Список рассылки будет создан и появится в дереве. Внутри списка рассылки автоматически создаются два раздела: "Адресаты" и "Ящики в Интернет":



В разделе "Адресаты" создаются почтовые адреса, на которые будут пересылаться все письма, отправляемые на адрес списка рассылки (в данном примере на адрес all@mydomain.ru). Читайте далее о работе с ними.

В разделе "Ящики в Интернет" создаются почтовые ящики Интернет, привязанные к этому списку рассылки.

Почтовые ящики Интернет в основном используются в связке с локальными почтовыми ящиками, о чем описано в разделе "Работа с почтовыми ящиками Интернет". Однако можно привязать несколько почтовых ящиков Интернет и к списку рассылки. В этом случае почтовый сервер будет скачивать всю почту из этих ящиков и сразу же рассылать по всем адресам, которые заведены в разделе "Адресаты".

Работа с адресатами списка рассылки


В списке рассылки можно создать любое количество адресов. Все письма, отправляемые на адрес списка рассылки будут автоматически пересылаться по всем адресам, заведенным в разделе "Адресаты" этого списка.

Создание нового адресата списка рассылки


1. Нажмите правой кнопкой мыши по значку или наименованию раздела "Адресаты" списка рассылки и выполните команду "Новый адресат" из контекстного меню.



2. Заполните поля в форме нового адресата



В поле "Адресат" указывается почтовый адрес. Это может быть как адрес почтового ящика, заведенного на данном сервере, так и любой другой адрес в Интернет.

В поле "Список рассылки" указывается адрес списка рассылки, к которому относится этот адресат. Изначально в этом поле находится адрес списка рассылки, в котором этот адресат создавался. Однако можно изменить его на другой. В этом случае адресат переместится в этот другой список рассылки.

Если включить флажок "Редактировать запись в адресной книге", то после нажатия на кнопку "ОК" появится окно, с помощью которого можно будет создать запись в адресной книге для этого адреса. Подробнее о работе с адресной книгой читайте в разделе "Работа с адресной книгой".

3. Нажмите кнопку "ОК". Адресат будет добавлен в список адресатов этого списка рассылки.

Изменение параметров адресата списка рассылки


Для изменения адреса в списке рассылки или для переноса адресата в другой список рассылки, нажмите левой кнопкой мыши по значку или наименованию этого адресата или нажмите правой кнопкой мыши и выполните команду "Изменить" из контекстного меню:



Измените имя адресата (его адрес) и/или список рассылки, в который он входит и нажмите "ОК".

Удаление адресата из списка рассылки


Для удаления адресата из списка рассылки нажмите правой кнопкой мыши по его значку или наименованию в дереве и выполните команду "Удалить" из контекстного меню.



Далее нужно подтвердить намерение удалить этого адресата и он будет удален из списка рассылки.

Изменение параметров списка рассылки


Можно изменить адрес списка рассылки или почтовый ящик, в который будут попадать письма, которые будут оцениваться почтовым сервером как спам при отправке по этому списку. Для этого нужно нажать левой кнопкой мыши по значку или имени нужного списка рассылки или нажать правой кнопкой мыши и выполнить команду "Изменить" из контекстного меню.



Далее в появившемся окне измените адрес списка рассылки и/или почтовый ящик для спама и нажмите "ОК".

Удаление списка рассылки


Для удаления списка рассылки нажмите на его значке в дереве или в наименовании правой кнопкой  мыши и выполните команду "Удалить" из контекстного меню.



После этого нужно подтвердить свое намерение удалить список, после чего список будет удален вместе со всеми его адресатами.
