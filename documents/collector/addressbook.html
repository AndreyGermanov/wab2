Работа с адресной книгой


Адресная книга позволяет хранить различную справочную информацию об адресах электронной почты. В адресной книге могут быть любые адреса: и которые зарегистрированы на почтовом сервере и любые другие. По умолчанию у каждой записи адресной книги для каждого адреса можно указать три поля: "Фамилия", "Имя" и "Отчество", но система управления позволяет изменить состав стандартных полей всех записей адресной книги и даже состав полей каждой записи.

В дальнейшем, можно использовать адресную книгу при рассылке почты по разным адресам, подставляя в письма наименования полей, заключенные в фигурные скобки. Например, если отправить письмо следующего содержания:

Здравствуйте {Имя} {Отчество} !

по адресу, запись с которым есть в адресной книге, то слова "{Имя}" и "{Отчество}" будут заменены на значения соответствующих полей этой записи и получатель получит уже измененное письмо.

Определение стандартных полей адресной книги


Для того чтобы изменить состав стандартных полей адресной книги, нужно нажать левой кнопкой мыши по значку или надписи "Адресная книга" в дереве. Появится список полей, которые будет иметь каждая запись адресной книги по умолчанию.



Строки таблицы "Поля адресной книги" редактируются. Редактируя строки этой таблицы вы изменяете состав стандартных полей адресной книги.

Вы можете перемещаться по строкам таблицы сверху вниз с помощью клавиши "Enter" и в обратном направлении с помощью комбинации "Alt+Enter"

Вы можете добавить строку в конец таблицы, перейдя в последнюю строку и нажав "Enter".

Вы можете вставить строку перед текущей, нажав комбинацию "Alt+Insert"

Вы можете удалить текущую строку, нажав комбинацию "Alt+Delete"

Вы можете перемещать текущую строку вверх и вниз по таблице с помощью комбинаций "Alt+<стрелка вверх>" и "Alt+<стрелка вниз>".

Чтобы отказаться от последней операции, нажмите "Esc".

Отредактировав состав полей нужным образом, нажмите клавишу "OK". Окно адресной книги закроется.

Добавление записи в адресную книгу


1. Нажмите правой кнопкой мыши по значку или надписи "Адресная книга" и выполните команду "Новый адрес" из меню.

2. Появится окно новой записи адресной книги.



Укажите в поле "Имя" адрес электронной почты и заполните значения полей адресной книги для него.

Изначально появляется стандартный набор полей, который мы указали ранее для всей адресной книги, однако для каждой конкретной записи вы можете определять любые поля, редактируя таблицу, с помощью приемов, описанных выше, при определении стандартных полей адресной книги.

3. После заполнения таблицы нажмите "OK".

Новый адрес появится в дереве и наименования полей можно использовать в качестве переменных типа "{Имя-поля}" в письмах, отправляемых по этому адресу.

Изменение записи адресной книги


В любой момент можно изменить состав и значение полей каждой конкретной записи адресной книги, в том числе и сам адрес. Для этого нужно нажать левой кнопкой мыши по значку или наименованию записи адресной книги в дереве или нажать правой кнопкой мыши и выполнить команду "Изменить" из контекстного меню.



Появится окно записи адресной книги, в котором можно изменить значения любых полей, как описано выше и нажать кнопку "ОК" для сохранения изменений.

Удаление записи адресной книги


Для удаления записи адресной книги нужно нажать правой кнопкой мыши на значке или адресе этой записи и выполнить команду "Удалить" из контекстного меню.
После этого потребуется подтвердить намерение удалить эту запись после чего она будет удалена из дерева.

Пример практического использования адресной книги


Адресная книга может быть полезна для проведения массовых рассылок. Например, вы хотите делать периодические рассылки по списку с большим количеством адресатов, но при этом, в письме здороваться с каждым адресатом персонально. Для этого нужно создать список рассылки и завести в нем всех адресатов, которые будут ее получать. Затем для каждого адресата нужно создать запись в адресной книге, и заполнить для них такие поля как "Фамилия", "Имя" и "Отчество". Все!

Теперь поясним как это будет работать на примере. Допустим вы создали список рассылки, который называется clients@mydomain.ru. В этом списке много адресатов, среди которых есть alserg777@mail.ru и vasil180984@rambler.ru . Также вы создали для всех адресатов записи в адресной книге, в том числе, для этих двух адресатов указали следующее:

Поле
 alserg777@mail.ru
 vasil180984@rambler.ru
Фамилия
 Петров
 Иванов
Имя
 Алексей
 Василий
Отчество
 Сергеевич
 Петрович

Теперь, если вы отправите по адресу clients@mydomain.ru письмо следующего содержания:

Здравствуйте {Имя} {Отчество} !

Приглашаем Вас принять участие в розыгрыше призов !

это письмо будет разослано всем адресатам списка clients@mydomain.ru, при этом по адресу alserg777@mail.ru придет письмо следующего содержания:

Здравствуйте Алексей Сергеевич !

Приглашаем Вас принять участие в розыгрыше призов !

а по адресу vasil180984@rambler.ru придет письмо следующего содержания:

Здравствуйте Василий Петрович !

Приглашаем Вас принять участие в розыгрыше призов !

При проведении периодических рассылок по большой базе адресов эта возможность может оказаться очень полезной.

Имейте в виду что замена происходит только в текстовой части письма, при условии что текст создается в кодировке UTF-8 (в большинстве почтовых клиентов это настраивается). Замена текста во вложениях не производится.

Также, не стоит делать такие рассылки с подстановками с помощью старых почтовых клиентов, таких как Microsoft Outlook Express, которые не работают с кодировкой UTF-8 и отправляют текстовую часть письма в формате 'quoted printable'.
