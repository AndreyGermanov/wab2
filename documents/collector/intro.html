Введение

Mystix Collector MX позволяет вам организовать собственное почтовое отделение, которое может использоваться как внутри организации для обмена почтой между сотрудниками в рамках локальной сети, так и в качестве полноценного почтового сервера в Интернет, обслуживающего любое количество почтовых доменов и почтовых ящиков в них.

Mystix Collector MX позволяет сконцентрировать всю почту организации внутри одного сервера, загружая ее из любых почтовых ящиков Интернет. С его помощью можно создавать списки рассылки, позволяющие автоматически рассылать письма сотням адресатов, автоматически заполняя эти письма данными из адресной книги.

Сервер содержит трехуровневую систему защиты от вирусов и спама, благодаря чему в почтовые ящики будет попадать только чистая почта, а спам будет отправляться в ящик для спама (в мусорную корзину), причем, у каждого пользователя может быть своя "мусорная корзина" для спама.

Mystix Collector MX настраивается с помощью удобной панели управления, доступ к которой можно получить с помощью браузера Mozilla Firefox по защищенному протоколу HTTPS. Также можно подключиться к серверу в режиме командной строки и удаленного рабочего стола.

В этом руководстве вы найдете необходимую инфомацию для понимания того, как работает почтовый сервер, как запустить его и настроить для решения различных задач.

В следующем разделе описан принцип работы почтового сервера, его взаимодействия с клиентами и другими почтовыми серверами, дан обзор всех его основных возможностей. Рекомендуем начать чтение с него. После этого раздела идут разделы с конкретными инструкциями по подключению, настройке и обслуживанию сервера.
