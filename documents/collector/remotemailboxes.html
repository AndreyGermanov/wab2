Работа с почтовыми ящиками Интернет


Механизм "почтовых ящиков Интернет" позволяет сконцентрировать всю почту пользователя в одном ящике. Благодаря этой возможности, почтовый сервер Mystix Collector MX в некоторых публикациях называют "почтовым концентратором". Смысл этого в следующем. У пользователя, имеющего почтовый ящик на сервере, также может быть несколько почтовых ящиков на различных серверах в Интернете (например в mail.ru или в yandex). Можно привязать эти почтовые ящики Интернет к почтовому ящику, заведенному на сервере и почта, приходящая на эти ящики будет автоматически скачиваться на почтовый сервер, в его почтовый ящик.

Если комплексно использовать эту технологию, она позволяет сконцентрировать всю почту компании на одном сервере.

Создание почтового ящика Интернет


1. Найдите в дереве почтовый ящик, к которому нужно привязать почтовый ящик Интернет

2. Нажмите правой кнопкой мыши по значку этого ящика или по его наименованию и выполните команду "Новый почтовый ящик Интернет" из контекстного меню.



3. Заполните поля формы, необходимые для подключения к серверу, на котором находится почтовый ящик Интернет, с которого нужно забирать почту



В поле "Имя" указывается адрес почтового ящика Интернет. Далее указывается "Владелец", то есть почтовый ящик на нашем сервере, к которому привязывается данный ящик. Далее указываются параметры доступа к серверу, на котором находится почтовый ящик Интернет: "протокол", "порт", "сервер", "пользователь" и "пароль". 

По умолчанию флажок "Активен" включен. Это значит, что почтовый сервер будет проверять этот почтовый ящик и загружать почту с него. Если нужно приостановить это, выключите флажок "Активен".

Если включить флажок "Редактировать запись в адресной книге", то после нажатия на кнопку "ОК" появится новая окно, позволяющее создать для этого адреса запись в адресной книге. Подробнее о работе с адресной книгой читайте в разделе "Работа с адресной книгой".

4. После ввода всех данных, нажмите "OK".

Почтовый ящик Интернет добавится в дерево и будет привязан к локальному почтовому ящику, для которого создавался. Если он активен, то каждые пять минут почтовый сервер будет подключаться к серверу с этим почтовым ящиком, используя указанные для этого параметры, скачивать с него всю новую почту и помещать в локальный почтовый ящик, к которому этот почтовый ящик Интернет привязан.

Можно для одного локального почтового ящика создать несколько почтовых ящиков Интернет. Таким образом можно добиться того, чтобы вся почта пользователя в итоге хранилась в одном ящике. В некоторых случаях это может быть довольно удобно, например, для целей резервного копирования.

Изменение почтового ящика Интернет


Можно изменить параметры подключения к почтовому ящику Интернет, а также его владельца, если, например, нужно сделать так, чтобы почта с этого адреса загружалась в другой почтовый ящик на сервере.

Чтобы изменить параметры почтового ящика Интернет, нажмите левой кнопкой мыши по его значку или наименованию в дереве или нажмите правой кнопкой мыши и выполните команду "Изменить" из контекстного меню.



Появится окно с параметрами почтового ящика Интернет, работа с которым была рассмотрена выше. Если нужно переместить этот почтовый ящик Интернет в другой локальный почтовый ящик, измените поле "Владелец".

Удаление почтового ящика Интернет


Для удаления почтового ящика Интернет нажмите по его значку или наименованию правой кнопкой мыши и выполните команду "Удалить" из контекстного меню.



После этого нужно будет подтвердить намерение удалить почтовый ящик Интернет. После этого почтовый ящик Интернет будет удален и исчезнет из дерева.
