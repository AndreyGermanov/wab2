Работа с почтовыми ящиками


В каждом почтовом домене можно создать любое количество почтовых ящиков. Почтовый ящик имеет адрес и папку на диске, внутри хранилища (/var/spool/mail), в которой хранится почта, приходящая по этому адресу. Чтобы получить доступ к содержимому почтового ящика, необходимо ввести имя пользователя и пароль. Эти и другие параметры нужно ввести при создании почтового ящика.

Создание почтового ящика


1. Нажмите правой кнопкой мыши по имени домена, внутри которого хотите создать почтовый ящик

2. Выполните команду "Новый почтовый ящик" из контекстного меню

3. Заполните форму нового почтового ящика



Закладка "Параметры":
  • Имя: адрес почтового ящика, который одновременно является и логином пользователя, для доступа к нему. Состоит из двух частей, разделенных символом @.Вторая часть имени это домен, в который входит почтовый ящик
  • Пароль: пароль пользователя почтового ящика. Требуется для доступа к почте из почтового клиента по протоколу POP3/IMAP и для отправки почты в Интернет по протоколу SMTP со своего почтового адреса. Пароль необходимо ввести дважды
  • Обратный адрес - адрес, который будет подставляться в каждое письмо в качестве обратного вместо адреса отправителя. Можно указать любой адрес. Его будет видеть получатель писем в качестве обратного.
  • Ящик для спама - ящик, в который будут пересылаться все приходящие по этому адресу письма, которые будут классифицированы почтовым сервером как спам. Ящик для спама это обычный почтовый ящик, который должен быть предварительно заведен на сервере.
  • Редактировать запись в адресной книге - если нажать этот флажок, то при сохранении изменений появится окно для ввода данных в адресную книгу для данного адреса. Подробнее о работе с адресной книгой читайте в разделе "Работа с адресной книгой".
Закладка "Автоответчик"


  • Включить автоответчик - включает автоответчик на этом почтовом ящике
  • Текст автоответчика - текст письма, которое будет отправляться получателю автоматически. При создании текста используется текстовый редактор, позволяющий оформить текст. Текст письма автоответчика сохраняется в формате HTML.

Если автоответчик включен, то почтовый сервер будет автоматически отвечать на любое письмо (кроме спама), которое будет приходить по этому адресу. В качестве текста письма будет использоваться текст, введенный в поле "Текст автоответчика".

После правильного заполнения формы нажмите "ОК", новый почтовый ящик будет создан на сервере и появится в дереве, внутри домена.  Почтовый сервер с этого момента будет принимать письма, отправляемые по адресу, указанному в поле "Имя:" и помещать их в почтовый ящик, связанный с этим адресом. Если вы заполнили поле "Почтовый ящик для спама", то часть писем, которые почтовый сервер будет классифицировать как спам, будут автоматически отправляться в этот почтовый ящик для спама. Если вы включили автоответчик, то каждому, кто отправляет письмо по этому адресу, почтовый сервер будет автоматически отвечать указанным текстом автоответчика.

Почтовый сервер также сможет принимать почту из Интернет, отправляемую на этот ящик. На самом сервере для этого не требуется делать никаких дополнительных настроек. Для этого нужно чтобы у сервера был статический IP-адрес и чтобы к этому статическому адресу было привязано имя домена, в котором находится этот почтовый ящик. Это прописывается на внешнем DNS-сервере (либо своем собственном, либо у провайдера или регистратора доменных имен). Также в DNS-сервере для этого домена должна быть создана MX-запись, указывающая что основным почтовым сервером данного домена является именно этот сервер.

Пользователь может настроить свой почтовый клиент, такой как Mozilla Thunderbird или Outlook Express для доступа к почте, хранящейся в этом почтовом ящике и для отправки почты с этого почтового ящика. При настройке почтовых клиентов потребуется ввести указанные здесь полное имя почтового ящика (в качестве имени пользователя), пароль и IP-адрес сервера (или доменное имя, если клиент может преобразовать это доменное имя в IP-адрес почтового сервера).

Клиент может отправлять почту с этого адреса как на другие адреса, заведенные в почтовом сервере, так и в Интернет. Для того чтобы почтовый сервер правильно находил почтовые сервера в Интернете, на которые отправлять почту, нужно правильно настроить DNS на нем (см. "Основные параметры сервера"). Также, чтобы почтовые сервера в Интернете принимали почту, отправляемую этим сервером, а не блокировали ее, желательно чтобы у него был внешний статический IP-адрес, и чтобы с этим статическим адресом было связано доменное имя. Иначе системы безопасности большинства почтовых серверов будут блокировать почту, которую передает этот сервер. Правильно настроенные почтовые сервера не принимают почту, которую отправляют с динамических или "голых" (без доменного имени) IP-адресов.

Изменение параметров почтового ящика


Можно изменить имя почтового ящика, пароль для доступа к нему и все остальные параметры, которые указывались при его создании. Для этого нужно нажать левой кнопкой мыши по имени или значку этого почтового ящика в дереве, либо нажать правой кнопкой мыши и выполнить команду "Изменить" из контекстного меню.
Появится окно параметров почтового ящика, работа с которым была описана выше. Измените нужные параметры и нажмите "ОК". Изменения применяются моментально. Потребуется перенастроить почтовых клиентов с учетом изменений.

Удаление почтового ящика
Для удаления почтового ящика нужно нажать правой кнопкой мыши по его значку или имени в дереве и выполнить команду "Удалить" из контекстного меню.
После этого нужно подтвердить свое намерение удалить почтовый ящик. После подтверждения почтовый ящик вместе со всей почтой в нем будет удален.
