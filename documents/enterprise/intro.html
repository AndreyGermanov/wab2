Введение

LVA Enterprise Server это сервер, организующий среду для построения виртуальной инфраструктуры организации. Он сам не выполняет функций какого-либо сервера, но позволяет создавать внутри себя любые сервера, например файловые, почтовые, Web-сервера, сервера БД и т. д. Среда управления LVA Enterprise Server использует расширения аппаратной виртуализации, встроенные в процессоры Intel для разделения аппаратных ресурсов между виртуальными машинами, которыми она управляет, что позволяет эмулировать оборудование со значительно меньшими издержками, чем при использовании чистых эмуляторов, таких как QEMU или VirtualPC. Система, основная задача которой заключается в управлении виртуальными машинами, называется гипервизором. Еще иногда ее называют "системой хоста",  она выполняет роль "хостинга виртуальных машин". Системы, установленные в виртуальные машины называют "гостевыми системами", они находятся под управлением "системы хоста". Этой терминологии мы будем придерживаться в данном руководстве.

LVA Enterprise Server выполняет роль гипервизора для гостевых систем виртуальных машин. Жесткий диск сервера разбит на два раздела. На первом разделе установлена операционная система и средства управления ей и виртуальными машинами которые в ней установлены. Второй раздел примонтирован к каталогу /data. В нем находятся виртуальные машины. Все они собраны в подкаталоге VirtualBox. Основаная задача LVA Enterprise Server заключается в том чтобы запускать эти машины, останавливать, следить за их работой и предлагать системному администратору удобный интерфейс для выполнения этих действий.

Обычно LVA Enterprise Server поставляется с предустановленными виртуальными машинами для решения различных задач. В качестве виртуальных машин могут выступать такие продукты как контроллер LVA Business Server или почтовый сервер Mystix Collector MX. В комплекте с каждым из этих продуктов есть панель управления, но если они поставляются в виде виртуальных машин, то их панели управления выносятся  из виртуальных машин в систему хоста в виде модулей единой панели, предназначенной для управления как самим гипервизором, так и виртуальными машинами. Эта панель управления является единой точкой входа в среду управления виртуальной инфраструктурой. С помощью этой панели можно запускать модули управления всеми компонентами системы хоста и виртуальных машин. Это могут быть интерфейсы управления, разработанные ЛВА, либо сторонние модули, такие как Webmin или phpVirtualBox.
